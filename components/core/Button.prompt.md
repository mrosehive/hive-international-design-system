Slate-led button with restrained gold emphasis — use for primary actions, form submits, and CTAs.

```jsx
<Button variant="primary" size="md" onClick={submit}>Request Briefing</Button>
```

Variants: `primary` (gold fill — keep scarce, one per view), `secondary` (outlined slate, border warms to gold on hover), `ghost` (gold text only), `danger` (outlined red). Sizes `sm` / `md` / `lg`. Labels render UPPERCASE with wide tracking. Pass `iconLeft` / `iconRight` (e.g. a Lucide icon node) for icon buttons.
