The "By the Numbers" metric — a large gold Bebas numeral over an uppercase label and supporting line.

```jsx
<StatBlock value="40+" label="Countries Served" detail="Cross-border operations" />
```

Use in multi-column rows; set `divider` on the 2nd+ columns to draw the thin gold vertical rule between metrics. `align="center"` for centered stat bands. Keep figures real — no invented data.
