Charcoal panel on slate — the default container for content blocks, capability tiles, and feature panels.

```jsx
<Card accent>
  <Eyebrow>Capability</Eyebrow>
  <h3>Digital Forensics</h3>
  <p>Evidence handling and intelligence-informed modernization.</p>
</Card>
```

`accent` adds the signature 3px gold left-rule (callouts/info panels). `hexTexture` overlays the faint honeycomb mesh for hero/feature cards — keep it off behind dense text. `interactive` adds a hover lift. Default padding 24px.
